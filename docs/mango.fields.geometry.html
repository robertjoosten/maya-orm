mango.fields.geometry module¶

class mango.fields.geometry.NurbsCurveField(*args, **kwargs)¶

Bases: mango.fields.base.Field

The NurbsCurveField is used to store nurbs curve data. This nurbs curve data can be used to generate actual nurbs curves in the scene. The return value will be an OpenMaya.MObject that holds reference to kNurbsCurveData it can be used to initialize a OpenMaya.MFnNurbsCurve object.

default_value¶

Used by autodoc_mock_imports.

get_parent_plug(plug)¶

When the plug is connected for nurbs curves we can’t use the connected plug as that is only the output plug. In this case we find the node and use its ‘create’ attribute to create the correct OpenMaya.MPlug instance.

get_plug_value(plug)¶

A copy of the data is made to make sure that the data object is not accidentally used to alter the data of the curve on the attribute. If any edits are made the object can be used to reset the attribute.
