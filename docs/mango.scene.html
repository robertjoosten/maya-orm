mango.scene module¶

mango.scene.initialize()¶

The initialization of a current scene find any object with a mango attribute and initialize its class. This will place it in memory and speed up the loading times. If the initialization of the model fails a warning message will be presented to the user.

mango.scene.migrate()¶

Types are stored by name in memory but the module is stored in the mango attribute so the class doesn’t have to be in memory for it to work as it gets imported when needed. When class’ move it is possible for this module path to be incorrect. This function will loop over all models initialized in the scene and reset the mango attribute and validate its assigned fields and relations.
